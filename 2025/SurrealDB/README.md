# 2025 challenges solved using SurrealDB database

This directory contains solutions to the 2025 Advent of Code challenges completed using [SurrealDB](https://surrealdb.com/). Code was written and executed using [Surrealist 3.6.8](https://surrealdb.com/surrealist) in a built-in sandbox. Should be possible to run it fully in a browser.
